Dessert Shop

An online dessert store featuring a persistent shopping cart, responsive design, and seamless user experience powered entirely by vanilla web technologies.

View on GitHub Live Demo

Project Overview

Dessert Shop is a fully responsive online store designed for a dessert business. The project focuses on delivering a smooth shopping experience using only vanilla web technologies — no frameworks, no dependencies.

The standout feature is its persistent shopping cart built with JavaScript and the Web Storage API (LocalStorage). Users can add items to their cart, close the browser, and return later to find their selections still intact. This creates a seamless experience that mimics professional e-commerce platforms.

The design is fully adaptive, ensuring a beautiful and functional experience on desktops, tablets, and smartphones. Clean CSS animations and interactive elements make the store feel alive and engaging.

Key Features

Persistent Shopping Cart

Cart data is stored in LocalStorage, allowing users to close the browser and return to find their items still saved. No accounts or backends required.

Fully Responsive Design

Adaptive layout that works beautifully on all screen sizes — from large desktop monitors to smartphones, ensuring accessibility for all users.

Product Catalog & Filtering

Organized product display with category filtering, making it easy for customers to browse and find their favorite desserts quickly.

Smooth Animations

CSS animations and transitions create an interactive, engaging experience with hover effects, cart updates, and page transitions.
