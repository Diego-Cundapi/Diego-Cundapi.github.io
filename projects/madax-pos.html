Project Overview

MADAX POS is a full-featured Point of Sale system built from the ground up for Refaccionaria MADAX, a real auto parts store. The system was designed to replace manual processes with a digital solution that handles inventory management, sales transactions, and business reporting.

Built with Laravel as the backend framework and packaged as a desktop application using NativePHP, the system provides a seamless experience for store employees. The application features role-based access control to ensure that each user only has access to the functionalities relevant to their position.

The system processes real transactions daily, demonstrating its reliability and stability in a production environment. It has significantly improved the store's operational efficiency by automating stock updates, generating instant PDF reports and receipts, and providing clear visibility into inventory levels.

Key Features

Role-Based Access Control

Comprehensive permission system ensuring each employee accesses only the features relevant to their role, enhancing security and operational control.

Inventory Management

Real-time stock tracking with automatic updates on every sale. Low stock alerts and detailed product cataloging for efficient management.

Sales Processing

Fast and intuitive sales interface with product search, quantity management, and automatic price calculations for quick checkout.

PDF Reports & Tickets

Automated generation of sales receipts, inventory reports, and financial summaries in professional PDF format.
